# 3D Solar System Simulation – [Your Name]

## Description
This is a 3D simulation of the Solar System built using Three.js. It includes:
- Realistic Sun and planets
- Adjustable orbit speeds per planet
- Pause/Resume animation
- Fully responsive canvas view

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern browser.
3. Use the control panel to adjust orbit speeds and pause/resume the animation.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- [Three.js](https://threejs.org/)

## Features
- All 8 planets with realistic distances, sizes, colors
- Real-time speed adjustment for orbits
- Interactive pause/resume button
